# MANEX

Proyecto de Alex Bernardos Gil, Covadonga Blanco Álvarez y Eneas de la Rosa Menéndez Pedrosa.

Orientado a la gestión de empleados, donde los mismos, pueden consultar días libres, horarios, vacaciones y gestiones como solicitar equipo.
También el rol de empleador puede ver la información y añadir nuevos empleados.
